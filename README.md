## 🛠️ Sobre o Workflow de Tradução

Algumas dessas traduções foram criadas utilizando um **workflow automatizado** em Python, desenvolvido para otimizar o processo de localização de jogos feitos na Unreal Engine. Este workflow foi fundamental para agilizar e padronizar o trabalho em muitos dos projetos listados.

Para mais detalhes sobre este workflow, suas ferramentas e como ele funciona, visite o repositório dedicado:

**[Unreal Engine Translation Workflow - Repositório](https://github.com/FinuraBR/UE-Translation-Workflow)**
